# Анализ маркетплейса Olist (Бразилия) 🇧🇷 📊

## 📌 О проекте
В данном репозитории представлен комплексный анализ данных крупнейшего бразильского маркетплейса Olist. Проект охватывает путь от обработки сырых данных с помощью SQL до построения интерактивного бизнес-дашборда в Power BI.

Цель проекта: Оценить динамику роста выручки, выявить проблемные зоны в логистике и проанализировать удержание клиентов (Retention) для принятия управленческих решений.

---

## 🛠 Технический стек
* Базы данных: PostgreSQL (сложные аналитические запросы, CTE, Window Functions).
* Визуализация: Power BI (DAX, создание связей, интерактивное картографирование).
* Инструменты: Power Query (ETL-процессы), Markdown.

## 📈 Ключевые этапы и инсайты

### 1. Финансовые показатели (Sales & Payments)
* MoM Growth: Реализован расчет темпов роста выручки месяц к месяцу. Выявлен пиковый рост в начале 2017 года (более 6000%), связанный с масштабированием платформы.
* Payment Efficiency: Анализ показал, что Credit Card является лидером по среднему чеку ($163.32), что на 12% выше, чем у Boleto.
* Вывод: Поддержка рассрочки через кредитные карты — ключевой фактор высокого оборота.

### 2. Логистический аудит (Logistics)
* Speed of Delivery: На интерактивной карте Бразилии визуализированы задержки доставки. 
* Инсайт: Большинство регионов работают эффективно, доставляя заказы на 8–19 дней раньше обещанного срока.
* Вывод: Высокая надежность логистики позволяет маркетингу использовать «быструю доставку» как УТП (уникальное торговое предложение).

### 3. Продуктовая аналитика (Retention & LTV)
* Retention Rate: С помощью когортного анализа выявлено крайне низкое удержание (менее 1% во 2-й месяц).
* LTV: Кривая жизненного цикла клиента выходит на плато почти сразу после первой покупки.
* Вывод: Бизнес-модель носит транзакционный характер. Рекомендовано внедрение CRM-стратегии для стимуляции повторных продаж.

## 🚀 Как использовать проект
